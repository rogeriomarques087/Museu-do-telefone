Projeto: Museu do Telefone
Autor: Rogério Marques Pereira
Trilha: 1


 Apresentando o Projeto
Este projeto é um ambiente de Realidade Virtual temático desenvolvido 
no Unity 6.3 LTS com o Meta XR SDK. O ambiente simula um museu virtual 
dedicado à evolução dos telefones ao longo da história, desde os 
aparelhos mais antigos até os smartphones modernos.

 Contexto e Objetivos
O Museu do Telefone VR foi criado no contexto do Metaverso como um 
espaço educativo e cultural. O objetivo é permitir que o visitante 
"caminhe" pelo museu em realidade virtual e visualize a evolução 
tecnológica dos telefones de forma imersiva e interativa.
Contexto: Educação e Entretenimento.

 Processo de Criação e Dificuldades
O projeto foi desenvolvido seguindo as etapas:
1. Configuração do Unity 6.3 LTS e instalação do Meta XR SDK
2. Configuração do Build Settings para Android (Meta Quest)
3. Montagem do ambiente: piso, paredes e mostruários
4. Importação de 2 modelos 3D de telefones em formato FBX
5. Configuração de Collider e Grab Interactable nos objetos
6. Organização da hierarquia e nomenclatura dos objetos
7. Configuração do Skybox

Principais dificuldades:
- Instalação do Meta XR SDK (Asset Store travou, solução via Package Manager)
- Notebook lento devido ao peso do Unity, cai objeto adicionado ficava pior
- Perda de progresso por travamento do notebook
- Limitação de memória obrigou uso de modelos 3D mais simples
- Limitação de memória obrigou também o uso de apenas 2 modelos simples

 Tecnologias Utilizadas
- Unity 6.3 LTS (6000.3.13f1)
- Meta XR SDK v201
- Modelos 3D em formato FBX (Sketchfab / Free3D)
